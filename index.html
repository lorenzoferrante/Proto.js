Proto.js is a JavaScript framework for quickly create web pages, animations and interactions. With Proto.js you can define animations with the integration with Velocity.js, that is similar to jQuery but faster.

The framework is written in Coffeescript, an upgraded and more legible version of Javascript. You can code it in both languages.

I've been inspired by the great Framer.js

  • Download Proto.js folder
  • Edit app.js (add layers and animations)
  • Open index.html in your browser

Using Proto.js is very simple. It is based on layers that are like html div. You have lots of properties you can define. Here is an example of how to create a new layer:

layerA = new Layer('layerA', {
    width: 200,
    height: 100,
    backgroundColor: '#efd55a',
    borderRadius: 5,
    position: 'absolute',
    x: 200
  });

Using the keyword new, the framework create a new layer with all the properties you declared. Before declaring properties, you must have to name the layer. The layer's name is also its html id, so you will be able to access it later with transitions and animations. Here's what the code above creates in the html page:

<div id="layerA" style="width: 200px; height: 100px; position: absolute; left: 200px; border-radius: 5px; background-color: rgb(239, 213, 90);"></div>

There is also a BackgroundLayer that is similar to html body tag. Its properties are backgroundColor and image. The second sets the image as a backround-image. When declaring it, you doesn't need to specific the layer's name. Here is an example declaration:

// Create a BackgroundLayer
layerA = new BackgroundLayer({
    backgroundColor: '#fff',
    image: 'nature.png'
});

Now let's see some other layer's declaration examples:

// Create a layer and center it
layerA = new Layer('layerA', {
    width: 200,
    height: 100,
    center: 'both'
  });

With center: 'both' you told the framework to center horizontally and vertically the layer. There are 3 possible ways to center a layer:

  • center: 'x' that center it only horizontally
  • center: 'y' that center it only vertically
  • center: 'both' that center it both vertically and horizontally

With edit() function you can easily change layer's properties after declaration. Here is an example:

layerA = new Layer('layerA', {
    width: 200,
    height: 100,
    center: 'both'
  });

// Change 'center' and 'width' property
layerA.edit('center', 'x');
layerA.edit('width', 300);

With delete() function you can delete a layer, or rather set its CSS visibility to hidden. Let's do an other example:

layerA = new Layer('layerA', {
    width: 200,
    height: 100,
    center: 'both'
  });

// Delete layerA
layerA.delete();

Let's see what are the available properties for the moment:

  • property -> example

  • width -> 230

  • height -> 100
  • backgroundColor -> '#ff32ee'
  • position -> 'absolute or 'relative or 'fixed'
  • x (left) -> 20
  • y (top) -> 0
  • z (z-index) -> 11
  • borderRadius -> 30
  • boxShadow -> '2px 1px 10px 4px rgba(0,0,0,0.3)'
  • center -> 'both' or 'x' or 'y'
  • image -> 'cat.png'
  • visible -> true or false
  • opacity -> 0.5
  • scale -> 1.5
    WARNING: Note that the scale property doesn't work if layer has animations. Use this animation instead: layer.animate('scale', 2, 2000)

Now let's talk about animations and events. I decided to use Velocity.js instead of jQuery for its better performance and speed. It's very simple to implement an event. Below the list of the possible (for the moment) animations, and how to use them.

/* -- SYNTAX -- */
layer.animate(animation_name, animation_property, animation_duration)
/* -- EXAMPLE -- */
layerA.animate('x', 200, 2000)
layerB.animate('rotateY', 80, 1000)
layerB.animate('backgroundColor', '#ff00ff', 750)

Here's the list of event's name (updated continuously) :

  • opacity
  • top
  • bottom
  • x
  • y
  • translateX
  • translateY
  • rotateX
  • rotateY
  • rotateZ
  • width
  • height
  • backgroundColor
  • borderRadius
  • boxShadow
  • scale
