# Proto.js
Coffeescript framework for quickly create web pages and animations. Inspired by <a href="https://github.com/koenbok/Framer">Framer.js</a>.

## Usage
```coffeescript
layerA = new Layer 'layerA',
	width: '200px'
	height: '100px'
	backgroundColor: '#efd55a'
	borderRadius: '5px'
	position: 'absolute'
	x: '200px' 
```
